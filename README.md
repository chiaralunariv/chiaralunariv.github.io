# chiaralunariv.github.io

Source for my personal academic website, built with GitHub Pages using the built-in `jekyll-theme-minimal` theme (no build step needed — GitHub builds it automatically).

## Structure
- `_config.yml` — site title/description + theme
- `index.md` — homepage
- `research.md`, `publications.md`, `cv.md` — subpages

## To edit
Just edit the `.md` files directly on GitHub (pencil icon) or clone locally, edit, and push. Changes go live within a minute or two.
